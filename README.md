# QuotesE2C

执行后自动将程序所在目录下指定文件（txt、srt、ass、ssa、vtt、lrc）中的英文引号替换为中文引号（包括双引号和单引号）。使用前请备份原文件。

</br>使用前请确保文中不存在左右引号中英混用的情况，否则会导致左右引号错乱。
